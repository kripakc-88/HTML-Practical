# HTML indexs Assignment

## Description
This repository contains basic HTML indexs created as part of a web development assignment.
The indexs demonstrate fundamental HTML concepts such as page structure, forms, tables,
and semantic HTML5 elements.

## indexs Included
- **index 1:** Simple Web Page (headings, paragraphs, lists, links, image)
- **index 2:** HTML Form (text input, radio buttons, checkboxes, textarea, submit button)
- **index 3:** HTML Table (table headings, data rows, merged cells, caption, borders)
- **index 4:** Semantic HTML Elements (header, navigation, article, aside, footer)

## Files
- `index.html` – index 1
- `index2.html` – index 2
- `index3.html` – index 3
- `index4.html` – index 4

## Steps to Run the Project
1. Download or clone the repository.
2. Open any `.html` file in a web browser (Chrome, Edge, Firefox).
3. View the output in the browser.
